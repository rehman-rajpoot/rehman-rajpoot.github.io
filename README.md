# Rehman Rajpoot – Portfolio

Static personal portfolio site (HTML, CSS, Vanilla JS).

## Features
- Responsive layout
- Dark mode toggle (localStorage)
- Accessible semantic structure
- Project cards generated from a small JS array
- Contact form (front-end validation only)
- Social / SEO meta tags

## Customize
1. Replace `assets/profile.jpg` and add `assets/og-image.png` (1200x630).
2. Update `index.html` meta description, about text, contact email.
3. Edit `script.js` → `projectData` with real projects.
4. Add your CV as `assets/Rehman-Rajpoot-CV.pdf` (or remove button).
5. Run a Lighthouse audit and tweak.

## Run Locally
Just open `index.html` or use VS Code extension "Live Server".

## Deploy (GitHub Pages User Site)
Repository name MUST be `rehman-rajpoot.github.io`.
Push these files to the main branch. Pages will auto-serve at:
https://rehman-rajpoot.github.io
